# tldhunter
tldhunter is python tool for brute forcing TLD and find all registered domain of a company

## how to use

### step 1

```bash
git clone https://github.com/valaDevs/tldhunter.git
```
### step 2

```bash
cd tldhunter
```

### step 3

```bash
pip3 install -r requirements.txt
```
### step 4
time to hack :)

```bash
python3 tldhunter.py
```
